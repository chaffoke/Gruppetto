🔐 Code administrateur
Chaque personne choisit son profil au premier lancement (via le bouton 👤 en haut) : son propre nom (ne peut modifier que ses pronostics des étapes à venir), invité (lecture seule), ou administrateur (accès complet — points, bonus, équipes, réglages). Ce code protège l'accès au profil Administrateur. Laisse vide pour que n'importe qui puisse devenir administrateur sans code.
Zone sensible
Utilise ceci quand un nouveau Tour démarre (2027, 2028…) — les joueurs et coureurs saisis ci-dessus seront repris pour la nouvelle saison.
La compétition reste entièrement modifiable — seul son statut change (utile pour signaler qu'elle est finie sans encore l'archiver).
Une compétition archivée devient définitivement figée en lecture seule — plus aucune modification possible, par personne. Elle reste consultable pour toujours depuis Archives, sur l'écran d'accueil.
